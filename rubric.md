# Scoring Rubric (6 Dimensions, 0-2 each, /12 total)

Candidates: total ≥ 9. Priority: ≥ 10. Ideal: ≥ 11.

## 1. Automatability
- **2**: Core deliverable is text output (writing, data, summarization, email, reports, code, SOPs). AI can do 70%+.
- **1**: Mix; AI does 30-60%.
- **0**: Requires human presence (video calls as core, physical work, real-time responsive, licensed judgment).

## 2. Oversight
- **2**: Async, output-graded, no time-tracking, no daily check-ins, no camera-on meetings.
- **1**: Weekly sync meetings. Some deadlines but flexible. Occasional spot-check.
- **0**: Time-tracking software (Hubstaff, Time Doctor, camera surveillance). Daily standups. Micromanaged.

## 3. Pay
- **2**: $50-100k base. Moderate sweet spot.
- **1**: $35-50k or $100-130k.
- **0**: Under $35k (not worth it) or over $130k (usually high-stakes / high-oversight).

## 4. Remote authenticity
- **2**: "Fully remote," no location language, no office-visit requirement.
- **1**: "Remote-first with occasional travel" — usually fine but check.
- **0**: "Hybrid," "remote with quarterly in-office," or tied to a specific metro.

## 5. Stakes
- **2**: Very little. If you're 20% worse than a normal employee, nothing bad happens.
- **1**: Small real consequences — worse deliverables, client disappointment, not legal/safety.
- **0**: High stakes — legal, medical, financial, safety-critical, or client-facing where reputation damage compounds.

## 6. Time flexibility
- **2**: No set hours. "40 hours a week, when they happen." No core-hours overlap requirement.
- **1**: Core hours of 4-6 hours/day but otherwise flexible.
- **0**: Strict 9-5 in a specific timezone. Mandatory synchronous hours.

## Red flag keywords (auto-penalty)

- "Hubstaff" / "Time Doctor" / "monitoring software" → oversight 0
- "Daily standup" → oversight ≤ 1
- "Must be available during" / "core hours required" → flexibility ≤ 1
- "Client-facing" / "face of the company" → stakes 0
- "Licensed professional" / "certified" → automatability ≤ 1
- "Video calls" (as core responsibility) → automatability ≤ 1
- "In-office quarterly" → remote 1 (not 0, but flagged)

## Green flag keywords (auto-bonus)

- "Async-first" / "async communication" → oversight 2
- "Results-only" / "ROWE" → oversight 2
- "Fully remote" / "work from anywhere" → remote 2
- "Flexible hours" / "own your schedule" → flexibility 2
- "Independent contributor" / "solo ownership" → oversight +1
- "Documentation-heavy" / "writing-focused" → automatability +1
